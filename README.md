# 学生宿舍管理系统简介 python441

## 项目概述

学生宿舍管理系统是基于Python技术构建的一套完整解决方案，旨在提高宿舍管理的效率和准确性。

## 技术栈

- 后端：Python
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

1. 学生信息管理
   - 学生信息录入
   - 学生信息查询
   - 学生信息修改
   - 学生信息删除
2. 宿舍楼信息管理
   - 宿舍楼信息录入
   - 宿舍楼信息查询
   - 宿舍楼信息修改
   - 宿舍楼信息删除
3. 宿舍分配管理
   - 宿舍分配
   - 分配记录查询
4. 维修与投诉
   - 维修请求提交
   - 投诉处理
5. 系统管理
   - 用户权限管理
   - 数据备份与恢复

## 系统角色

- 系统管理员
- 宿舍管理员
- 学生

## 运行环境

- 操作系统：Linux/Windows
- Python版本：3.x
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库
3. 部署后端服务
4. 部署前端界面
